# 网络天才

通过一系列不同的问题，让玩家回答“是”、“否”、“不知道”、“或许是”或“或许不是”，以缩小符合条件的人物范围，最后询问玩家它所猜出的这个人物是否正确。

## 使用说明

由于中国大陆境内无法访问akinator网站，所以必须在`utils.py`中填写代理地址，否则插件将无法使用

## 指令说明

|指令|说明|
|-----|-----|
|网络天才|开始游戏|
|结束网络天才|结束游戏|
|是/不是/可能是/可能不是/不知道|回答问题|
|b|返回上一个问题|
